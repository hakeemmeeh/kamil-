# Dev scripts

One-off utilities for sourcing images and IDs. Not part of the Next.js app.

| Script | Purpose |
|--------|---------|
| `get_unsplash_ids.py` | Scrape Unsplash photo IDs from search pages |
| `get_unsplash_ddg.py` | DuckDuckGo image search helper |
| `get_unsplash_google.py` | Google image search helper |
| `get_wiki_images.py` | Wikipedia image lookup |
| `test_ids.py` | Validate Unsplash photo IDs |

Run from repo root, e.g. `python3 scripts/get_unsplash_ids.py`.
