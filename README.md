# Kamil Travel

A premium, modern web application for Kamil Travel, providing services such as car rentals, airport transfers, corporate travel, destinations, tours, and visa assistance.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- Next.js 15
- React 19
- Tailwind CSS
- GSAP for animations
- Lucide React for icons
